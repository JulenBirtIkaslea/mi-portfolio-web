# Mi Portfolio Web

Este proyecto es un portfolio web básico creado como práctica de Git y GitHub.

Se ha desarrollado partiendo de un repositorio local inicializado con Git, conectándolo posteriormente a un repositorio remoto en GitHub y siguiendo un flujo de trabajo basado en commits estructurados (feat, fix, chore, docs).

El proyecto incluye:

- Estructura HTML5 básica
- Archivo de estilos CSS enlazado correctamente
- Configuración de `.gitignore`
- Corrección de contenido mediante commit de tipo `fix`
- Documentación del proyecto en este README

El objetivo principal ha sido practicar el flujo completo de trabajo con Git: inicialización, conexión con remoto, control de versiones, commits semánticos y despliegue en GitHub.
